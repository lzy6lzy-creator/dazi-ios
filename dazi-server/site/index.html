为什么靠谱

它不是随机推荐，而是先判断合不合适。

i搭不搭会把活动需求拆成时间、地点、活动类型和偏好，再让双方 AI 先聊一轮。你看到的不是一堆陌生人，而是更可能一起成行的人。

时间合得上 优先

不把根本碰不上的人推给你，活动时间先过一遍。

地点不折腾 优先

附近、同城、区域表达都会被理解，尽量少走冤枉路。

活动偏好接近 优先

比如安静看展、轻松徒步、不尬聊，细节也会被纳入判断。
